# 财务智析 Web 原型

这是一个无需安装依赖的静态 Web 财务分析软件原型，覆盖资产负债表、利润表、现金流量表、偿债能力、盈利能力、营运能力、综合能力、杜邦分析和分析报告。

## 运行

直接打开 `index.html` 即可查看。也可以在本目录启动本地服务：

```powershell
python -m http.server 5173
```

然后访问 `http://localhost:5173`。

## 当前能力

- 三期演示财务数据联动分析
- KPI 仪表盘与评分体系
- 资产结构、利润趋势、现金流瀑布图
- 偿债、盈利、营运、综合能力指标页
- 杜邦分析树
- 报告预览、打印导出 PDF、文本报告下载
- 三大报表 CSV / JSON / Excel 导入
- 中文科目与英文字段自动映射
- 导入后自动刷新期间选择、KPI、图表和报告
- 三大报表 CSV 模板下载
- 行业基准对比与指标详情抽屉
- 风险预警中心与 AI 财务点评
- 多公司横向对比
- 情景预测：收入增长率、毛利率、费用率、回款天数
- 报告编辑器：导出前可编辑管理层摘要
- 本地登录：首次创建账号，之后使用本地账号密码登录
- 个人中心：头像上传、资料编辑、偏好设置、密码修改、本地数据概览

## 导入说明

在“报表导入”页面分别上传资产负债表、利润表、现金流量表。推荐先点击页面右侧的“资产模板 / 利润模板 / 现金流模板”下载 CSV 模板。

Excel 导入使用浏览器端 SheetJS CDN。若本地网络阻止 CDN 加载，CSV / JSON 导入仍可正常使用。

Excel 会优先按工作表名称自动匹配资产负债表、利润表、现金流量表；没有匹配名称时读取第一个工作表。

## 登录说明

首次打开系统会要求创建本地账号。账号数据保存在当前浏览器的 `localStorage` 中，密码保存为 SHA-256 哈希，不保存明文。退出后需要重新输入账号密码登录。

## 个人中心

登录后可在左侧进入“个人中心”。头像、显示名称、岗位、部门、联系方式、界面偏好和默认首页都会按当前账号保存到本地。头像以 Base64 图片数据保存在浏览器中，建议上传 2MB 以下图片。
